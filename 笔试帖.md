# 8.2拼多多笔试

- 求运行结果：
```javascript
var b = 10;
(function b(){
    b = 20; // window.b = 20
    window.b = 1;
    console.log(b);
})();
```

结果：

```javascript
function b(){
    b = 20;
    window.b = 1;
    console.log(b);
}
```

> 首先，因为函数声明先于变量声明
